# Sensor REST API 

Backend-сервис для IoT-платформы экологического мониторинга. Приложение предназначено для приема, обработки и хранения метрик с внешних датчиков в режиме реального времени.

## Основной функционал
* Мониторинг температуры и влажности в помещениях.
* Агрегация прогнозов погоды и отслеживание магнитных бурь.
* Отказоустойчивый REST API для взаимодействия с фронтенд-клиентом и аппаратной частью.

## Технологический стек
* **Язык:** Java
* **Фреймворк:** Spring Boot
* **Архитектура:** REST API
* **База данных:** [PostgreSQL]

## ⚙️ Особенности реализации
* Интеграция потоковых данных с внешнего оборудования.
* Оптимизированная структура БД для быстрого агрегирования метрик.
